# AI Disclosure Statement

## Project Title

London Airbnb Market Intelligence Platform

## Purpose of Disclosure

This document provides transparency regarding the use of Artificial Intelligence (AI) tools during the development of this project.

The project was completed by the author, with AI tools used as supportive resources for learning, debugging, documentation, and development guidance.

---

## AI Tools Used

The following AI tools were used during project development:

* ChatGPT (OpenAI)
* Claude

---

## Areas Where AI Assistance Was Used

AI tools were used for the following purposes:

### Development Guidance

* Understanding data engineering concepts and workflows.
* Learning best practices for data cleaning and feature engineering.
* Receiving guidance on machine learning model development and evaluation.

### Debugging Support

* Troubleshooting Python errors and exceptions.
* Resolving Streamlit dashboard implementation issues.
* Fixing model integration and deployment problems.

### Statistical Analysis Support

* Understanding statistical testing methodologies.
* Reviewing effect size calculations and regression outputs.

### Documentation Assistance

* Structuring project documentation.
* Improving readability of findings and reports.


### Dashboard Design Guidance

* Improving dashboard layout and user experience.
* Designing business-oriented analytics views.
* Organizing navigation and presentation components.

---

## Author Validation

All AI-generated suggestions, code snippets, explanations, and recommendations were manually reviewed and validated before inclusion in the project.

The author:

* Executed all code independently.
* Verified data transformations and outputs.
* Trained and evaluated machine learning models.
* Interpreted statistical findings.
* Tested dashboard functionality.
* Made final implementation decisions.

AI tools were used as assistants and learning aids rather than as autonomous project creators.

---

## Academic Integrity Statement



AI tools were used responsibly to support learning, development, debugging, and documentation. All final deliverables and implementation decisions were reviewed, validated, and approved by the author.
